## 识五线谱练习工具
这个应用面向五线谱初学者，提供一个五线谱片段和一个虚拟的88键钢琴，用户在正确点击了片段中每个音符在琴键上的正确位置时可以得分；

### 技术方案
这是一个SPA的工具应用
- React + Vite + Typescript + TailwindCss
- 业务需求关键依赖库推荐方案：
  - `vexflow`：用于浏览器端五线谱渲染，支持音符、谱号、节拍等乐谱元素；本项目已采用。
  - `Tone.js`（可选）：若后续需要“按键发声”“节拍器”“听音反馈”，可引入该库管理 WebAudio 时序与合成器。
  - `zustand`（可选）：当后续加入训练模式、题库、难度分级和历史统计时，可将训练状态集中管理，降低组件耦合。

当前实现说明：
- 已实现：五线谱片段展示、88 键钢琴点击、按目标音符判分、回合推进、连击与准确率统计。
- 当前练习范围：自然音（C4-B5），适合初学者做基础读谱训练。
